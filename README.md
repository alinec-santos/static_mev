# static_mev
Para testar o detector personalizado: 
1- ativar o venv :dentro de static_mev/src dar o comando -  source venv/bin/activate
2- Em seguida entrar na pasta detectors e dar o comando : pip install -e . - para instalar o plug-in do detector
3- Em seguida sair da pasta detector e dar o comando- python3 run_all.py
